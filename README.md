# Xtrec Website

A clean, modern static website for Xtrec - a UK-based technology company building smart connected hardware and software products.

## Structure

```
xtrec-site/
├── index.html          # Homepage
├── products.html       # Products page
├── platform.html       # Platform & Innovation page
├── sustainability.html # Sustainability page
├── contact.html        # Contact page
├── style.css           # Stylesheet
└── README.md           # This file
```

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern Typography**: Uses Outfit (headings) and Manrope (body) fonts
- **Minimal & Professional**: Deep blue (#002E5D) accent colour on white backgrounds
- **SVG Logo**: Custom logo with connected nodes concept
- **Mobile Navigation**: Hamburger menu for smaller screens
- **Contact Form**: Ready for backend integration

## Pages

1. **Homepage** (`index.html`)
   - Hero section with headline and product silhouettes
   - Three product pillars (Carbon Platform, Ambient Displays, Sports Devices)
   - "Why Xtrec" highlights section
   - Call-to-action for pilot programme

2. **Products** (`products.html`)
   - Detailed information on each product line
   - Problem/solution format
   - Feature lists
   - "Pilot users welcome" badges

3. **Platform** (`platform.html`)
   - Shared IoT platform explanation
   - Platform benefits and capabilities
   - Technical architecture overview
   - Partnership CTA

4. **Sustainability** (`sustainability.html`)
   - Net-zero mission
   - Scope 1, 2, 3 emissions explanation
   - Company commitments
   - Stats and impact metrics

5. **Contact** (`contact.html`)
   - Contact form with interest selection
   - Company contact information
   - Pilot programme benefits
   - "What happens next" steps

## Usage

Simply open `index.html` in a web browser, or deploy to any static hosting service:

- GitHub Pages
- Netlify
- Vercel
- AWS S3
- Any web server

## Brand Guidelines

- **Primary Color**: #002E5D (Deep Blue)
- **Background**: #FFFFFF (White)
- **Alt Background**: #F8FAFC (Light Gray)
- **Text**: #0F172A (Dark Slate)
- **Muted Text**: #64748B (Gray)
- **Headings Font**: Outfit
- **Body Font**: Manrope
